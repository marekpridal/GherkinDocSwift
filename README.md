# GherkinDoc

Doc generator for Gherkin steps written in Swift.

## Usage

Your StepDefinition file should have sections and comments. Checkout ExampleStepDefinition.swift for better understanding. If you have your StepDefinition file marked down, open terminal, run ./make.sh to compile generator and then run `GkerkinDoc` command. Insert path to your StepDefinition file and that's it ðŸŽ‰.
